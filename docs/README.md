# Tobii IS5平台 DIY 眼动仪

### Step 0. 前置要求

- 电脑必须是 ***Winodws*** 系统

- 系统不能是精简版

- 如果你发现我有东西漏发了，及时联系我即可

- （如需Windows Hello人脸识别）登录用户名不能是默认的**Administrator**账户

### Step 1. 设备连接

首先，需要明确两个概念：

1、所有眼动仪都不支持27寸以上屏幕，因为屏幕越大追踪效果越差

2、所有屏幕﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠式眼动仪都需要安装在屏幕下方，这款不一定贴在屏幕下边缘，但是需要靠近屏幕下边缘



概念清楚后，那么我们就可以使用包装里附带的磁吸条和支架进行设备的固定了
﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠
根据屏幕类型，请选择合适的固定方法：

#### Step 1.1 台式机配普通直屏/笔记本/笔记本外接直屏显示器

普通台式机直屏直接使用包装内的磁吸铁片进行固定即可

撕掉后面3M胶的背膜，横着贴在屏幕下面的边框上

![显示器引磁片安装位置示例图](https://s2.loli.net/2025/12/04/8rtMnEmylqKP3cU.png)

如果你是笔记本，那么这条黑色的贴片可能会太厚，会影响到笔记本内屏开合，使用包装内的银色贴片即可（如果忘记发的话联系我，或直接淘宝购买，[链接在这里](https://item.taobao.com/item.htm?_u=239t9l649f15&id=680784574872&skuId=5314893098112)）

贴到下图转轴附近Logo位置即可

![笔记本引磁片粘贴位置示例](https://s2.loli.net/2025/12/04/XIJsubG1U3472rR.png)

#### Step 1.2 曲面屏

曲面屏则会因﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠为屏幕弧度原因无法使用直条的引磁片

这个时候就需要使用曲屏支架

将引磁片贴在曲屏支架较细的一面，这样子设备即可磁吸上

另一面贴上附送的3M双面胶，并贴在显示器下部，如下图所示（演示的是直屏，曲屏同理贴上一部分即可）

![曲屏支架安装示例图](https://s2.loli.net/2025/12/04/zweEc6fgrRjxmB3.jpg)



---

然后，连接上附带的Type-C数据线即可，推荐**使用一字螺丝刀把固定的螺丝拧紧**



### Step 2. 驱动安装

首先需要进行告知：

**Tobii家的驱动出了名的问题多**，特别是外星人笔记本的定制驱动问题更多，就算是硬破后与Eye Tracker 5保持一致，新手的平均安装时长也会达到**`40min`**左右



这也是为什么我只卖硬破后的版本，不卖那种没有破解过需要使用特定的驱动的版本（也就是同行3xx的那个），这﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠个版本曾经以1xx价格卖过一段时间，约40单，统计数据为：`90%`顾客需要远程协助，`95%`的电脑会出现安装问题，`10%`的电脑需要整体重装后才能正常使用，且多屏使用时稳定性极差。

**所需准备：**

- Tobii Eye Tracker 5 官方驱动安装包（点击[这里](https://gaming.tobii.com/getstarted/?bundle=tobii-et5))进去下载页面或点击[这里](https://files.update.tech.tobii.com/Tobii.EyeTracker5.Offline.Installer_4.183.0.30025.msi)直接下载1.133版本安装包）

- 一个能访问微软商店的网络

- 微软商店（如果是工作站版等没有自带微软商店的版本，可以百度搜索安装教程进行安装）

### Step 3 校准

在进行这一步前，请确认设备表面的**保护膜**已经撕除，否则会导致校准失败

进入驱动后，首先会要求你进行屏幕校准，屏幕校准方法如下：

首先，拖动屏幕上的﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠两条竖线，对准设备外壳的左右两端

![屏幕尺寸校准第一步](https://s2.loli.net/2025/12/04/QyWpBnIHx8g2dt4.jpg)

然后，拖动屏幕里的眼动仪模型，将其最右侧位置对准设备的最右侧Type-C接口位置

![屏幕尺寸校准第二步](https://s2.loli.net/2025/12/04/LbHeoKBp8kdVErh.jpg)

这是为了让系统准确计算你的屏幕大小和偏移值﻿⁠⁠﻿⁠﻿⁠﻿⁠⁠⁠⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿⁠⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠﻿﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠⁠⁠﻿﻿﻿﻿﻿⁠⁠⁠﻿﻿⁠⁠﻿⁠⁠﻿﻿﻿﻿⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠﻿﻿﻿⁠⁠﻿⁠﻿⁠﻿⁠﻿﻿﻿﻿⁠﻿⁠⁠﻿﻿⁠⁠﻿⁠⁠⁠﻿⁠⁠⁠﻿﻿﻿⁠⁠⁠﻿﻿﻿﻿⁠⁠⁠⁠⁠⁠⁠﻿⁠﻿﻿﻿⁠﻿﻿⁠⁠﻿⁠⁠﻿⁠﻿⁠⁠⁠，保证精度



然后即可点击创建档案进行眼动追踪的校准。
